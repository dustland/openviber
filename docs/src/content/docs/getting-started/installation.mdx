## Install Viber

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install viber
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add viber
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add viber
    ```
  </TabItem>
</Tabs>

## Requirements

- **Node.js**: 18.x or later
- **TypeScript**: 5.0+ (recommended)
- **Framework**: React 18+, Svelte 5+, or vanilla JavaScript

## Environment Setup

Viber requires API keys for the LLM providers you want to use:

```bash
# .env
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...
```
