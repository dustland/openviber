## Quick Example

```typescript
import { Agent, streamText } from 'viber';

const agent = new Agent({
  name: 'Assistant',
  model: 'openai:gpt-4o',
  systemPrompt: 'You are a helpful assistant.',
});

const result = await agent.streamText({
  messages: [{ role: 'user', content: 'Hello!' }],
});
```
