## Create Your First Agent

<Steps>

1. **Import Viber**

   ```typescript
   import { Agent, Space, streamText } from 'viber';
   ```

2. **Define an Agent**

   ```typescript
   const agent = new Agent({
     name: 'Assistant',
     model: 'openai:gpt-4o',
     systemPrompt: 'You are a helpful assistant.',
   });
   ```

3. **Stream a Response**

   ```typescript
   const result = await agent.streamText({
     messages: [{ role: 'user', content: 'Hello!' }],
   });

   for await (const chunk of result.textStream) {
     process.stdout.write(chunk);
   }
   ```

</Steps>

## Framework Integration

<Tabs>
  <TabItem label="React">
    ```tsx
    import { useAgent } from 'viber/react';

    function Chat() {
      const { messages, sendMessage } = useAgent({
        model: 'openai:gpt-4o',
      });

      return (
        <div>
          {messages.map((m) => (
            <p key={m.id}>{m.content}</p>
          ))}
        </div>
      );
    }
    ```
  </TabItem>
  <TabItem label="Svelte">
    ```svelte
    <script lang="ts">
      import { createAgentStore } from 'viber/svelte';

      const agent = createAgentStore({
        model: 'openai:gpt-4o',
      });
    </script>

    {#each $agent.messages as message}
      <p>{message.content}</p>
    {/each}
    ```
  </TabItem>
</Tabs>
